# Web Application Security Testing Checklist

A comprehensive checklist for manual penetration testing of web applications, aligned with OWASP Testing Guide v4.

## Categories

- **Information Gathering** — Crawling, directory enumeration, tech stack fingerprinting
- **Authentication Testing** — Default credentials, brute-force, password policy, MFA bypass
- **Authorization Testing** — IDOR, privilege escalation, forced browsing
- **Session Management** — Cookie flags, session fixation, CSRF
- **Input Validation** — SQLi, XSS, command injection, file upload
- **Business Logic** — Workflow bypass, parameter tampering, race conditions

## Usage

Use as a tracking sheet during assessments. Check off items as tested.
